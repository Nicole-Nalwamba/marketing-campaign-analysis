# Maven Marketing Campaign Analysis 📊

This project is part of my Data Analytics portfolio and serves as the final capstone of my learning journey. It involves analyzing a real-world marketing campaign dataset using **Power BI** to uncover customer behavior, product performance, and campaign effectiveness.

## 📁 Project Overview

The goal of this project is to help Maven Marketing make informed decisions by analyzing:

- Customer demographics and profiles
- Product category performance
- Marketing campaign success rates
- Channel effectiveness
- Factors influencing web purchases

## 💡 Key Insights

- The average customer is ~44 years old, married, and has a university education.
- Wines and meat products are top-selling product categories.
- Campaign 3 performed best; Campaigns 1 and 5 underperformed.
- Web purchases are influenced by income, education, and age.
- Catalog and deal channels are underutilized.

## 📊 Tools Used

- **Power BI** for data visualization and interactive dashboards
- **DAX** for calculated columns and measures
- **Data Cleaning** for preparing customer and campaign data
- **Segmentation** via age groups and product categories

## 🧠 Skills Demonstrated

- Data Cleaning and Transformation
- Data Modeling and Relationships
- DAX Measures and Calculated Columns
- Dashboard Design and Data Storytelling
- Business Insight Communication
